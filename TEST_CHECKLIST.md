## Quick Test Commands

### Test Locally
1. Open `index.html` in browser
2. Press `F12` to open DevTools
3. Check Console for errors
4. Use "Toggle Device Toolbar" (Ctrl+Shift+M) for mobile view

### Test Animations
1. Scroll up and down multiple times
2. Hover over all cards
3. Click all buttons
4. Resize browser window

### Test Links
1. Right-click each link
2. Select "Open in new tab"
3. Verify it goes to correct page

## Common Issues & Fixes

### Profile Image Not Showing
- ✅ Check file name is exactly `profile.jpeg`
- ✅ Check file is in `images/` folder
- ✅ Refresh browser (Ctrl + F5)

### Animations Not Working
- ✅ Check JavaScript console for errors
- ✅ Ensure `script.js` is loaded
- ✅ Try different browser

### Links Not Working
- ✅ Check URLs are correct
- ✅ Ensure links start with `https://`
- ✅ Test in incognito mode

### Mobile Menu Not Opening
- ✅ Check hamburger icon is visible
- ✅ Click directly on icon
- ✅ Check JavaScript console
