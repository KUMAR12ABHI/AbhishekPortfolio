# Quick Setup Guide 🚀

## Step 1: Prepare Your Images

1. **Save your profile photo** (the one you uploaded) as `profile.jpg` in the `images/` folder
2. **Download or screenshot your certificates** and save them in the `images/` folder with these exact names:
   - `java-cert.jpg` - HackerRank Core Java
   - `sql-basic.jpg` - HackerRank SQL Basic
   - `sql-intermediate.jpg` - HackerRank SQL Intermediate
   - `jse-cert.jpg` - JSE Certificate
   - `fullstack-cert.jpg` - Full Stack Certificate
   - `internship-jee.jpg` - Internship JEE Certificate
   - `java-interview.jpg` - Java Interview Module

## Step 2: Add Your Resume PDF

Copy your resume PDF as `KUMAR_ABHISHEK_Resume.pdf` in the portfolio folder.

## Step 3: Test Locally

1. Open `index.html` in your browser
2. Check if everything looks good
3. Test all links (LinkedIn, GitHub, certificates, etc.)

## Step 4: Deploy to GitHub Pages

### Create GitHub Account (if you don't have one)
1. Go to https://github.com
2. Sign up for free

### Upload to GitHub

1. Create a new repository named `portfolio`
2. Upload all files from the portfolio folder
3. Go to Settings → Pages
4. Select branch: `main`, folder: `/ (root)`
5. Click Save
6. Wait 2-3 minutes
7. Your site will be live at: `https://YOUR_USERNAME.github.io/portfolio/`

## Step 5: Share Your Portfolio

Add your portfolio link to:
- LinkedIn profile
- Resume
- Email signature
- GitHub profile README

---

## Need Help?

If you face any issues:
1. Check that all image names match exactly
2. Ensure `index.html` is in the root of your repository
3. Wait a few minutes after enabling GitHub Pages
4. Clear your browser cache

## Pro Tips

- Keep image sizes under 500KB for faster loading
- Use JPG format for photos
- Test on mobile devices
- Update content regularly
- Add Google Analytics (optional)
