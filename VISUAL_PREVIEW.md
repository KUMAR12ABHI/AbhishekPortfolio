## 🎬 Animation Preview

### When Page Loads:
1. **0.0s** - Particles start floating
2. **0.5s** - "Hi, I'm" fades in
3. **0.8s** - "Kumar Abhishek" types out with gradient
4. **1.5s** - "Java Backend Developer | SDE-2" types with cursor
5. **2.5s** - Cursor stops blinking and disappears
6. **0.5s** - Profile image fades in and starts floating
7. **1.0s** - Buttons fade in
8. **1.2s** - Social icons fade in

### While Scrolling:
- Sections fade in from bottom
- Section titles animate underline
- Stats count up when visible
- Cards reveal with stagger effect

### On Hover:
- **Buttons**: Glow and lift
- **Skill Cards**: Lift, scale, shine effect
- **Achievement Cards**: 3D tilt, shine animation
- **Certificate Cards**: 3D tilt, overlay appears
- **Social Icons**: Lift and change color
- **Profile Image**: Slight zoom
